# OI 班主任模拟器 — 当前状态

> 最后更新：2026-06-13

## 项目概述

一个交互式 Web 游戏，玩家扮演信息学竞赛班班主任，带领学生从 NOIP 冲到 NOI，为期两年（64 周）。每周最多做 3 件事（教学、管理、关系维护等），面对随机事件和比赛压力。

| 拆分版 | `workspace/oi-teacher-simulator/` | index.html + game.js + style.css，6/7 实验性拆分，未完全同步 |

## 已完成功能

- [x] 核心循环：64 周 × 每周 3 行动
- [x] 5 个学生档案（各有算法/数学/心态/体力属性）
- [x] 8 种行动选项（讲课、专题算法、CF刷题、心理疏导等）
- [x] 随机事件系统（40+ 事件，含 CCF 政策、机器故障、凡尔赛等）
- [x] 比赛系统（NOIP、省选、NOI 等，带成绩计算）
- [x] 成就系统（13 个成就）
- [x] 比赛结果模态框（展示每人表现）
- [x] 自动存档（开始、周结束、事件处理后）
- [x] 周过渡动画
- [x] 结局画面（多种结局 + 统计）
